# Xray-node

A collection of scripts to deploy high-performance proxy services on various container platforms including Node.js & Docker.

## Features

- ** Tri Protocol Support**: Xray Vmess,VLESS & Trojan.
- **Deployment**: Docker environments

## Supported Protocols
- *Xray + Argo Tunnel*

## Deployment Targets

- **Node.js Container Platforms*
